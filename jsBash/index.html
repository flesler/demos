Help

  • Commands can be: Most Array, String, jQuery and Math object methods.
    Math and binary operators, 'set' returns the argument, '()' runs a function.
    'lambda' creates a lambda function from a string, and optional arguments.
    'map' and 'each' work on array/hash, check the commands on the right.
    'dump' shows the output, accepts -c(console) or -d(DOM), else alert()
  • Arguments can be: Literal arrays, strings, numbers, regexes or hashes.
    Content wrapped with parens will be eval'd, flags can be used (-x,-y ...)
    % % define lambda functions on the fly, which arguments are a,b,c,d.
    The words window,document.body,document,true,false,null are identified.
    >foo sets the output of a command, into the variable foo.
    <foo gets the content of the variable, the pipe (|) connects commands.
